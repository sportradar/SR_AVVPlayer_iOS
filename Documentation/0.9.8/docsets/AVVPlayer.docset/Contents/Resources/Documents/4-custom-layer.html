Custom Layer

AVVPlayerCustomLayerDelegate

The AVVPlayer SDK provides default layers which are displayed on different states of the player. For example if the player runs into an error, a default error layer is displayed. There might be requirements where you need to provide your own error layer. Therefore you can implement AVVPlayerCustomLayerDelegate to inject your custom layer. Set players customLayerDelegate to force AVVPlayer to ask for a custom error layer when player runs into an error.

    player?.customLayerDelegate = self : nil

The following code block shows a prototype how you can implement a custom error Layer:

class MyCustomErrorLayer : UIView, AVVErrorLayer {

    var layerView: UIView!
    {
        return self
    }

    var error : AVVError?

    var action: AVVErrorAction?

    func reset() {}
}

When the player runs into an error and you’ve implemented and set the customLayerDelegate the errorLayer delegate method will be called. Return your custom error layer here.

func errorLayer(player: AVVPlayer, forError error: AVVError) -> AVVErrorLayer? {
    let errorLayer = MyCustomErrorLayer()
    errorLayer.error = error
    return errorLayer
}

By implementing AVVPlayerCustomLayerDelegate you can also provide your own loading, preview VOD/LIVE and externalPlayback layer.
