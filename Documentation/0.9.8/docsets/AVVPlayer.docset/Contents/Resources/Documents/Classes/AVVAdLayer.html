AVVAdLayer

public class AVVAdLayer : UIView

How is this class created?

The AVVMediaSession creates an instance of the AVVAdLayer and passes it to the AVVAdvertisementProvider

Usage

Use the adContainer of this class to display ads.

  • adContainer

    Load your ads into this view.

    Declaration

    Swift

    public var adContainer: UIView
  • startLoadingAnimation()

    Starts a UIIndicatorView in the center of the AVVAdLayer

    Usage:

    Call this function before you request an ad.

    Example:

    func request(adItems: [AVVAdvertisementItem], onCompletion: (() -> ())?) {
    adLayer?.startLoadingAnimation()
    // load your ads
}

    Declaration

    Swift

    public func startLoadingAnimation()
  • stopLoadingAnimation()

    Stops a UIIndicatorView in the center of the AVVAdLayer

    Usage:

    Call this function when an ad starts.

    Example:

    func adStarted() {
    adLayer?.stopLoadingAnimation()
}

    Swift
