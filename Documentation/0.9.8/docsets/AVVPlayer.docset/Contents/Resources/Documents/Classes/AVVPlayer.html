    Description

  • mediaSession

    Description

    The AVVMediaSession is responsible for playing the video. It handles the AVPlayer, ChromeCast, Ads, etc. and holds the information about them.

    Usage

    Subscribe on the mediaSession to get information about current player events, e.g.: playing, pause, volume change. You can subscribe on the mediaSession in the avvPlayer(_ player : AVVPlayer, initializesMediaSession session : AVVMediaSession) method.

    Warning

    This property is nil, when initializing the player.

    Declaration

    Swift

    public private(set) var mediaSession: AVVMediaSession? { get }

    Description

    Warning

    Description

    This method resets all layers, closes the current media session, and starts the player new.

    Usage

    Call this method to start your player. You do not have to call play() thereafter.
