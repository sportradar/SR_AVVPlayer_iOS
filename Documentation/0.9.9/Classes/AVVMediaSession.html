AVVMediaSession

public class AVVMediaSession : NSObject, AVVObservable, AVVMediaPlaybackControlable
extension AVVMediaSession: AVVChromeCastConnectionDelegate
extension AVVMediaSession : AVVAdvertisementProviderDelegate

Class which is responsible for playback of a given stream url. Holds a reference to AVVPlayerConfig to setup and control item related services (e.g. Ads). After playback if finished, AVVMediaSession should be deallocated.
