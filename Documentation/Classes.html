  • AVVPlayer

    The AVVPlayer Main Class

    control the player with this class

        // e.g.:
    player.play()
    player.pause()
    player.suspend()

    control the style of the player

        // e.g.:
    player.hideLayer(view: view, animated: true)
    player.insertLayer(view: view, forType: .customLayerAboveControls, animated: false)

    add your observers on this class

        // e.g.:
    class MyPlayerContainer: UIView, AVVPlayerObserver {
        init() {
            player.add(self)
        }
    }

    add your delegates

        // e.g.:
    class MyPlayerContainer: UIView, AVVPlayerCustomLayerDelegate, AVVPlayerConfigDelegate, AVVPreviewLayerDelegate {
        init() {
            player.customLayerDelegate = self
            player.configDelegate = self
            player.previewLayerDelegate = self
        }
    }
